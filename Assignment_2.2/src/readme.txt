
	This file contains 2 java programs: 
	
	1. GenoNucleicAcid.java creates 5 NucleicAcid objects and asks the user to input the Name, Chemical Formula, Molar Mass, and Density. 
		The program then prints the information of each Nucleic Acid.
	
	
	2. NucleicAcid.java contains the blueprint to create a NucleicAcid object and is instantiated in GenoNucleicAcid.java. The file contains a method to print information about each Nucleic Acid, 
		and also has methods that allow the user to access and change the Nucleic Acid's information (such as it's Name, Chemical Formula, Molar Mass, and Density)
