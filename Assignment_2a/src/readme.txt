
	This file contains 2 java programs: 
	
		1. Driver.java creates 3 person objects and asks the user to input their name, 
		   age, address, and phone number. The program then prints each person's information.
		
		
		2. Person.java initializes a person object created in Driver.java, and contains a method 
		   to print a person's information, and also methods that allow the user to access and change
		   a person's information(such as their name, age, address, and phone number).
