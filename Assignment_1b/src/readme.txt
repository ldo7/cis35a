
	This file contains 2 java programs: 
	
	1. ComputeLoan.java asks the user to input the initial loan amount, annual interest rate, and length of loan in terms of years.
		The program calculates the user's total payment after interest, and also how much they will pay per month towards interest 
		and their principal loan amount.
	
	
	2. SquareRootApproximation.java asks the user to input any positive number
		and then returns the square root.
